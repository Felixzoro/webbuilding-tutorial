Aufbau 1 - html

Hier wird kurz erklärt, wie der html code aufgebaut ist:

HTML ist die Sprache, die Webseiten beschreibt. Sie verwendet Tags, um Elemente wie Überschriften (h1), Absätze (p), Bilder (img) usw. zu definieren.

Jedes Element hat einen Anfangs- und einen Endtag (<>...< />), und man kann ihnen Attribute hinzufügen, die zusätzliche Informationen liefern.

HTML-Dokumente haben eine grundlegende Struktur mit einem DOCTYPE, einem HTML-Tag, einem Head-Tag für Metadaten und einem Body-Tag für den eigentlichen Inhalt.
